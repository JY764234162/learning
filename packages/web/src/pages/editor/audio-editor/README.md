# 音频编辑器

一个功能完整的在线音频编辑器，支持音频上传、波形可视化、剪切、调速、调音量等功能。

## 功能特性

### 📤 音频上传
- 支持点击上传和拖拽上传
- 支持多种音频格式：MP3、WAV、OGG、M4A 等

### 🌊 波形可视化
- 基于 WaveSurfer.js 实现高质量音频波形显示
- 实时显示播放进度
- 支持波形缩放

### ✂️ 音频剪切
- 通过拖拽创建选区
- 可调整选区大小和位置
- 支持导出选区为独立音频文件
- 可播放预览选区

### 🎛️ 播放控制
- 播放/暂停
- 音量调节（0-100%）
- 播放速度调节（0.25x - 2x）
- 实时时间显示

### 💾 导出功能
- 导出完整音频
- 导出选区片段
- 支持 WAV 格式导出

## 安装依赖

在项目根目录运行以下命令安装所需依赖：

```bash
pnpm add wavesurfer.js
```

或使用 npm：

```bash
npm install wavesurfer.js
```

## 技术栈

- **WaveSurfer.js**: 音频波形可视化和操作
- **Web Audio API**: 音频处理和导出
- **React Hooks**: 状态管理
- **Ant Design**: UI 组件库

## 使用说明

1. **上传音频**：点击或拖拽音频文件到上传区域
2. **播放控制**：使用播放/暂停按钮控制音频播放
3. **调整参数**：使用滑块调整音量、速度和缩放级别
4. **添加选区**：点击"添加选区"按钮，在波形上创建可编辑的选区
5. **调整选区**：拖动选区边缘调整范围，拖动选区移动位置
6. **导出音频**：
   - 点击选区的"导出"按钮导出该片段
   - 点击顶部"导出"按钮导出完整音频

## 快捷操作

- **双击波形**：跳转到指定位置
- **拖动选区边缘**：调整选区范围
- **拖动选区中间**：移动选区位置
- **点击选区外部**：取消选区选中状态

## 注意事项

1. 音频文件过大可能会导致加载时间较长
2. 导出的音频格式为 WAV（未压缩），文件会比较大
3. 建议在现代浏览器中使用（Chrome、Firefox、Edge 等）
4. 某些浏览器可能需要用户手势（如点击）才能播放音频
